# EditorPDF

Website for editing PDF files

Web-сайт для редактирования PDF файлов

Позволяет просматривать и вносить изменения в  существующие файлы pdf-докуметов.

Требования:

	1.	Возможность загрузки файлов с компьютера;
	2.	Возможность просматривать документы;
	3.	Возможность объединять несколько документов в один;
	4.	Возможность разделять один документ;
	5.	Возможность вставить пустую страницу;
	6.	Возможность удалить выбранную страницу;
	7.	Возможность переключения между загруженными документами;
	8.	Возможность вставки изображения на выбранную страницу файла;
	9.	Функция вставки нумерации страниц;
	10.	Возможность скачивания отредактированного файла на компьютер.

Требуемые элементы:

	1.	Главное окно;
	2.	Элемент просмотра документа;
	3.	Элементы управления функциями сайта.

Основания разработки:
